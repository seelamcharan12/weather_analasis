# 🌦️ Weather Data Analysis and Visualization

## 📌 Project Overview
This project focuses on analyzing and visualizing weather data using Python. The dataset is generated using NumPy and analyzed using Pandas. Various visualizations are created using Matplotlib and Seaborn to understand weather patterns.

---

## 🛠️ Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn

---

## 📊 Dataset Details
The dataset contains 100 records with the following features:
- Date
- Temperature (°C)
- Humidity (%)
- Wind Speed (km/h)
- Rainfall (mm)

The dataset is synthetically generated using NumPy.

---

## 📈 Key Analysis
- Average and maximum temperature
- Total rainfall calculation
- Monthly temperature trends
- Relationship between temperature and rainfall

---

## 📉 Visualizations
- Temperature trend over time (Line Chart)
- Humidity distribution (Histogram)
- Temperature vs Rainfall (Scatter Plot)
- Correlation heatmap
